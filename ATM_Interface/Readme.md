#  Project 03 - ATM Interface

##  Description

The ATM Interface is a console-based Java application that simulates basic banking operations using Object-Oriented Programming (OOP).

The application allows users to securely access their account using a PIN and perform common ATM operations.

---

##  Features

-  PIN Authentication
-  Deposit Money
-  Withdraw Money
-  Check Account Balance
-  Transaction History
-  Input Validation
-  Exit Option

## How to Run

1. Clone the repository.
2. Open the project in VS Code or any Java IDE.
3. Compile the program:
   ```bash
   javac DecodeLabs_Java_P3.java
   ```
4. Run the program:
   ```bash
   java DecodeLabs_Java_P3
   ```
