# Student Grade Calculator

## Description
A Java console application that calculates:
- Total Marks
- Average Marks
- Percentage
- Grade
- Pass/Fail Status

## Features
- Dynamic number of subjects
- Input validation
- Exception handling
- Grade calculation
- Pass/Fail determination

## Technologies Used
- Java
- Scanner
- Exception Handling
- Loops
- Conditional Statements

## How to Run
1. Compile:
   javac DecodeLabs_Java_P2.java

2. Run:
   java DecodeLabs_Java_P2
