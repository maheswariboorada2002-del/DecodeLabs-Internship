#  DecodeLabs Java Internship

##  Projects

###  Project 01 - Number Guessing Game
A console-based game where the user guesses a randomly generated number.

**Features**
- Random number generation
- User input handling
- Multiple rounds
- Score calculation
- Input validation

**Concepts Used**
- Java Basics
- Loops
- Conditional Statements
- Random Class
- Scanner
